```sh
# Web Components UI
$ npm i -S @xgqfrms/wcui

```

## usage

```html
<!-- import wcui -->
<script src="../dist/wcui/index.min.js"></script>

<!-- use wcui-??? -->
<wcui-toast content="this is one custom `Toast` components"></wcui-toast>

```
